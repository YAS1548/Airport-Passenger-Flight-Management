# Système de Gestion Aéroport (simple)

Projet de gestion des passagers et des vols développé en langage C
dans le cadre des travaux pratiques de L2 à l'USTHB.

## Fonctionnalités
- Gestion des vols (ajout, capacité, destination)
- File d'attente des passagers (Queue)
- Embarquement des passagers (Stack)
- Gestion dynamique de la mémoire (listes chaînées)

## Structures de données utilisées
- File (Queue) : liste d'attente des passagers
- Pile (Stack) : passagers embarqués
- Listes chaînées : stockage dynamique

## Compilation
gcc "Airport Passenger and Flight Management System.c" -o airport
./airport

## Langage
C (C99)
